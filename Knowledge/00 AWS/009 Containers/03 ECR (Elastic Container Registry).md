# What's ECR?
---

* A managed service that **stores and manage Docker images on AWS**
	* Fully **integrated with ECS**, backed by Amazon S3
	* Category
		* Private Repository
		* Public Repository (Amazon ECR Public Gallery https://gallery.ecr.aws)
